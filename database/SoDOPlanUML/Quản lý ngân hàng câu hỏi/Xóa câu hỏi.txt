@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

' Định nghĩa các thành phần
actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Xóa câu hỏi

== Truy cập ngân hàng câu hỏi ==
Actor -> UI: Chọn "Quản lý ngân hàng câu hỏi"
UI -> System: Yêu cầu lấy danh sách câu hỏi
System -> DB: Query danh sách câu hỏi
DB --> System: Trả kết quả dữ liệu
System --> UI: Hiển thị danh sách câu hỏi
UI --> Actor: Hiển thị giao diện danh sách

== Xác nhận xóa ==
Actor -> UI: Chọn "Xóa câu hỏi" (ID)
UI -> UI: Hiển thị Popup xác nhận xóa

alt không (Hủy xóa)
    Actor -> UI: Chọn "Không"
    UI -> UI: Thực hiện đóng Popup
else có (Xác nhận xóa)
    Actor -> UI: Chọn "Có"
    UI -> System: Gửi yêu cầu xóa câu hỏi theo ID
    System -> DB: Thực hiện xóa khỏi CSDL (DELETE)
    DB --> System: Xác nhận đã xóa thành công
    System --> UI: Trả về thông báo thành công
    UI --> Actor: Hiển thị thông báo xóa thành công
end

@endum
@enduml
