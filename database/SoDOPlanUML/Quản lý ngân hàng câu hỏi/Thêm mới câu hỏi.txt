@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

' Định nghĩa các thành phần
actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Thêm câu hỏi vào ngân hàng đề

== Xem danh sách ==
Actor -> UI: Chọn "Quản lý ngân hàng câu hỏi"
UI -> System: Yêu cầu lấy danh sách câu hỏi
System -> DB: Query danh sách câu hỏi
DB --> System: Trả kết quả dữ liệu
System --> UI: Hiển thị danh sách câu hỏi
UI --> Actor: Hiển thị giao diện danh sách

== Khởi tạo thêm mới ==
Actor -> UI: Click "Thêm câu hỏi"
UI -> System: Yêu cầu Form thêm mới
System --> UI: Trả về Form nhập liệu
UI --> Actor: Hiển thị Form thêm câu hỏi

== Lưu câu hỏi & Kiểm tra trùng lặp ==
Actor -> UI: Nhập nội dung câu hỏi và nhấn "Lưu"
UI -> System: Gửi nội dung câu hỏi mới
System -> DB: Truy vấn các câu hỏi hiện có
DB --> System: Trả về danh sách nội dung để so sánh
System -> System: Kiểm tra độ tương đồng (Similarity Check)

alt không có hoặc ít tương đồng
    System -> DB: Lưu câu hỏi mới vào CSDL
    DB --> System: Xác nhận lưu thành công
    System --> UI: Thông báo thành công
    UI --> Actor: Hiển thị thông báo thành công
else tương đồng trên 80% hoặc toàn bộ
    System --> UI: Thông báo thất bại (Câu hỏi đã tồn tại)
    UI --> Actor: Hiển thị thông báo thất bại
end
