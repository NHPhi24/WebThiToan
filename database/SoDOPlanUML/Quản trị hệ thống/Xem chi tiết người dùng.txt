@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

' Định nghĩa các thành phần
actor "Admin" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Xem chi tiết người dùng

== Truy cập danh sách ==
Actor -> UI: Chọn "Quản lý người dùng"
UI -> System: Yêu cầu lấy danh sách người dùng
System -> DB: Query danh sách người dùng
DB --> System: Trả kết quả dữ liệu
System --> UI: Hiển thị danh sách người dùng
UI --> Actor: Hiển thị giao diện danh sách

== Xem chi tiết ==
Actor -> UI: Chọn "Xem chi tiết" (ID)
UI -> System: Gửi yêu cầu xem chi tiết theo ID
System -> DB: Kiểm tra ID người dùng có tồn tại?
DB --> System: Kết quả (Có/Không)

alt có
    System -> DB: Truy vấn thông tin chi tiết
    DB --> System: Trả về dữ liệu chi tiết
    System --> UI: Trả về dữ liệu hiển thị Modal
    UI --> Actor: Hiển thị Modal thông tin chi tiết
else không
    System --> UI: Trả về thông báo lỗi
    UI --> Actor: Hiển thị thông báo thất bại
end

@endum
@enduml
