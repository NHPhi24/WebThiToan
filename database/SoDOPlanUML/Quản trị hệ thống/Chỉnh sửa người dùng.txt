' Định nghĩa các thành phần
actor "Admin" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Chỉnh sửa người dùng (Hiệu chỉnh Logic)

== Truy cập danh sách ==
Actor -> UI: Chọn "Quản lý người dùng"
UI -> System: Yêu cầu lấy danh sách người dùng
System -> DB: Query danh sách
DB --> System: Trả kết quả
System --> UI: Hiển thị danh sách
UI --> Actor: Hiển thị giao diện danh sách

== Khởi tạo Chỉnh sửa ==
Actor -> UI: Click "Chọn chỉnh sửa người dùng"
UI -> System: Yêu cầu Form & Dữ liệu chi tiết
System -> DB: Query người dùng theo ID
DB --> System: Trả dữ liệu hiện tại
System --> UI: Trả về Form chứa dữ liệu
UI --> Actor: Hiển thị Form chỉnh sửa

== Thực hiện cập nhật ==
Actor -> UI: Chỉnh sửa thông tin và nhấn "Lưu"
UI -> System: Gửi yêu cầu cập nhật
System -> DB: Cập nhật thông tin vào CSDL

alt đã tồn tại (Hợp lệ để cập nhật)
    DB --> System: Xác nhận thành công
    System --> UI: Thông báo thành công
    UI --> Actor: Hiển thị thông báo thành công
else không tồn tại (Lỗi dữ liệu)
    System --> UI: Thông báo thất bại (Người dùng không còn tồn tại)
    UI --> Actor: Hiển thị thông báo thất bại
end
