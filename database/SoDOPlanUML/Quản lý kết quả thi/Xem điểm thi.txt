@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Xem điểm thi

== Truy cập chức năng ==
Actor -> UI: Chọn "Quản lý kết quả thi"
UI -> System: Yêu cầu danh sách ca thi
System -> DB: Truy vấn danh sách ca thi
DB --> System: Trả về dữ liệu
System --> UI: Hiển thị nút chọn ca thi
UI --> Actor: Giao diện chọn ca thi

== Thực hiện xem điểm ==
Actor -> UI: Chọn ca thi cụ thể
UI -> System: Gửi yêu cầu lấy điểm (ID ca thi)
System -> DB: Kiểm tra ca thi có tồn tại không?
DB --> System: Kết quả kiểm tra

alt đã tồn tại
    System -> DB: Truy vấn danh sách điểm của học sinh đã nộp bài
    DB --> System: Trả về danh sách điểm
    System --> UI: Hiển thị danh sách điểm của học sinh đã nộp bài
    UI --> Actor: Danh sách kết quả thi chi tiết
else không tồn tại
    System --> UI: Hiển thị trang trống
    UI --> Actor: Thông báo không có dữ liệu
end
@enduml
