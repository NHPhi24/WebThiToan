@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

' Định nghĩa các thành phần
actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Thêm ca thi

== Truy cập danh sách ==
Actor -> UI: Chọn "Quản lý ca thi"
UI -> System: Yêu cầu lấy danh sách ca thi
System -> DB: Truy vấn danh sách ca thi
DB --> System: Trả kết quả dữ liệu
System --> UI: Hiển thị danh sách ca thi
UI --> Actor: Hiển thị giao diện danh sách

== Khởi tạo thêm mới ==
Actor -> UI: Chọn "Thêm ca thi"
UI -> System: Yêu cầu Form nhập liệu
System --> UI: Hiển thị Form nhập
UI --> Actor: Hiển thị Form (Tên ca, thời gian...)

== Lưu dữ liệu ==
Actor -> UI: Nhập thông tin ca thi và nhấn "Lưu"
UI -> System: Gửi yêu cầu lưu ca thi mới
System -> DB: Kiểm tra tên ca thi có tồn tại không?
DB --> System: Kết quả kiểm tra (Đã tồn tại / Không tồn tại)

alt không tồn tại
    System -> DB: Lưu thông tin ca thi vào CSDL (INSERT)
    DB --> System: Xác nhận thành công
    System --> UI: Trả về thông báo thành công
    UI --> Actor: Hiển thị thông báo thành công
else đã tồn tại
    System --> UI: Trả về thông báo lỗi (Tên ca thi đã tồn tại)
    UI --> Actor: Hiển thị thông báo thất bại
end
@enduml
