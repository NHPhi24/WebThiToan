@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

' Định nghĩa các thành phần
actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Xóa cấu trúc đề thi

== Điều hướng chức năng ==
Actor -> UI: Chọn "Quản lý đề thi"
UI -> System: Yêu cầu danh sách đề thi
System -> DB: Query danh sách đề thi
DB --> System: Trả kết quả
System --> UI: Hiển thị danh sách và nút thêm cấu trúc

Actor -> UI: Chọn "Thêm đề thi"
UI -> System: Yêu cầu Form và nút cấu trúc
System --> UI: Hiển thị Form và nút cấu trúc đề thi

Actor -> UI: Chọn "Cấu trúc đề thi"
UI -> System: Yêu cầu danh sách cấu trúc
System -> DB: Query danh sách cấu trúc
DB --> System: Trả kết quả
System --> UI: Hiển thị danh sách cấu trúc đề thi

== Xác nhận xóa ==
Actor -> UI: Chọn "Xóa" (ID cấu trúc)
UI -> UI: Hiển thị Popup xác nhận xóa

alt không (Hủy xóa)
    Actor -> UI: Chọn "Không"
    UI -> UI: Thực hiện đóng Popup
else có (Xác nhận xóa)
    Actor -> UI: Chọn "Có"
    UI -> System: Gửi yêu cầu xóa cấu trúc theo ID
    System -> DB: Thực hiện xóa khỏi CSDL (DELETE)
    DB --> System: Xác nhận xóa thành công
    System --> UI: Trả về thông báo thành công
    UI --> Actor: Hiển thị thông báo xóa thành công
end

@enduml
