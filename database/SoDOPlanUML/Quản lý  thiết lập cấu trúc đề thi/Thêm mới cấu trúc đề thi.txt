@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

' Định nghĩa các thành phần
actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Thêm cấu trúc đề thi

== Điều hướng chức năng ==
Actor -> UI: Chọn "Quản lý đề thi"
UI -> System: Yêu cầu danh sách đề thi
System -> DB: Query danh sách đề thi
DB --> System: Trả kết quả
System --> UI: Hiển thị danh sách đề thi

Actor -> UI: Chọn "Thêm đề thi"
UI -> System: Yêu cầu Form & Nút cấu trúc
System --> UI: Hiển thị Form và nút cấu trúc đề thi

== Thiết lập cấu trúc ==
Actor -> UI: Chọn "Cấu trúc đề thi"
UI -> System: Yêu cầu danh sách cấu trúc hiện có
System -> DB: Query danh sách cấu trúc
DB --> System: Trả kết quả
System --> UI: Hiển thị danh sách cấu trúc và nút thêm

Actor -> UI: Chọn "Thêm cấu trúc"
UI -> System: Yêu cầu Form nhập cấu trúc
System --> UI: Hiển thị Form nhập liệu

== Lưu dữ liệu ==
Actor -> UI: Nhập thông tin cấu trúc và nhấn "Lưu"
UI -> System: Gửi thông tin cấu trúc mới
System -> DB: Kiểm tra tên cấu trúc đã tồn tại?
DB --> System: Kết quả (Đã tồn tại/Chưa)

alt chưa tồn tại
    System -> DB: Lưu cấu trúc vào CSDL
    DB --> System: Xác nhận thành công
    System --> UI: Trả về thông báo thành công
    UI --> Actor: Hiển thị thông báo thành công
else đã tồn tại
    System --> UI: Trả về thông báo lỗi (Tên đã tồn tại)
    UI --> Actor: Hiển thị thông báo thất bại
end

@endum
@enduml
