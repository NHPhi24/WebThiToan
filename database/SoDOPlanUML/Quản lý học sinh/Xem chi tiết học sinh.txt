@startuml
autonumber
skinparam style strictuml
skinparam sequenceMessageAlign center

actor "Giáo viên" as Actor
boundary "Giao diện (UI)" as UI
control "Hệ thống" as System
database "DB" as DB

title Sơ đồ tuần tự: Xem chi tiết học sinh

== Truy cập danh sách ==
Actor -> UI: Chọn "Quản lý học sinh"
UI -> System: Yêu cầu lấy danh sách học sinh
System -> DB: Truy vấn danh sách học sinh
DB --> System: Trả kết quả dữ liệu
System --> UI: Hiển thị danh sách học sinh
UI --> Actor: Giao diện quản lý học sinh

== Thực hiện xem chi tiết ==
Actor -> UI: Chọn "Xem chi tiết học sinh"
UI -> System: Gửi yêu cầu xem chi tiết (ID học sinh)
System -> DB: Kiểm tra ID học sinh có tồn tại không?
DB --> System: Kết quả kiểm tra (Tồn tại/Không tồn tại)

alt đã tồn tại
    System -> DB: Lấy thông tin chi tiết học sinh
    DB --> System: Trả về dữ liệu chi tiết
    System --> UI: Hiển thị modal thông tin chi tiết
    UI --> Actor: Hiển thị thông tin học sinh đầy đủ
else không tồn tại
    System --> UI: Hiển thị thông báo thất bại
    UI --> Actor: Thông báo lỗi: Học sinh không tồn tại
end
@enduml
